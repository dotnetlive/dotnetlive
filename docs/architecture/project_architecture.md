## 共享项目(纯技术类，辅助其他项目)

* dotnetlive项目
我们网站的主站，其宣传与其他站得导流

* dotnetlive.account项目
负责登陆与浏览器端Cookie生成

* dotnetlive.admin项目
综合后台管理项目

* dotnetlive.authcenter
账户权限集中授权中心

* dotnetlive.logcenter
DNL项目集中日志查看中心

## 业务项目

* dotnetlive.blog
博客项目, 类似于博客园的多人博客项目

* dotnetlive.forums
论坛项目

* dotnetlive.exam
创建问题，组合考题，在线答题， 便于各种在线评测

* dotnetlive.career
一个简历管理站点，以及工作意向管理，业内各公司之间内推

* dotnetlive.ecommerce
DNL电商项目

* dotnetlive.cms
DNL CMS项目
